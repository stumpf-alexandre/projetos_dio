Itálico

(Voltar)

O itálico é quando os traços da caligrafia das palavras fica ligeiramente inclinado à direita. É usada para dar ênfase a palavra ou frases por fazer parte de um grupo de gírias, expressões estrangeiras, destacando nome de obras, entre outras.

A tag responsavel por transformar a palavra em itálico é o "i". A tag deve ser iniciada e fechada como demonstrado: "<i>exemplo</i>"

Negrito

(Voltar)

O negrito é utilizado para substituir o sublinhamento de palavras, ou seja destacar determinadas palavras dentro de um texto, dando assim uma forte importância.

A tag responsavel por modificar as palavras em negrito é o strong, e como o "i", a tag tem um inicio e um fechamento "<strong>exemplo </strong>"

Esses dois elementos do HTML podem ser usados simultaneamente elevando ainda mais a importância da palavra em destaque: "<strong><i>exemplo </i></strong>"

Input

(Voltar)

O input é um elemento do HTML que retorna um conjunto de dados digitados pelo usuário, na forma de Strings. Esses dados podem ser de um tipo especifico, basta acrescentar o elemento "type" na tag "input", com os valores number para receber números, text para receber uma frase, color para receber uma cor, email para receber um endereço de e-mail, entre outros.

Img

(Voltar)

A tag img representa a inserção de imagem na pagina que esta sendo criada.

Esta tag possui o atributo src="", que é por onde nós lincamos as imagen, seja ela de uma pasta especifica do programa ou seja ela uma imagem da internet.

H1, h2, h3...

(Voltar)

As tags h1, h2, h3, h4, h5, h6, são elementos que representam seis níveis de titulo de seção. Sendo que h1 pode ser para representar titulos, h2 para subtitulo, h3 para intertitulos e assim sucessivamente. Um elemento de cabeçalho descreve brevemente os tópicos da seção em que ele está.

Lista de Itens

(Voltar)

Ordenada

O elemento que representa no HTML uma lista de itens ordenados é o <ol></ol>, com pelo menos um elemento filho, representado pela tag <li></li>.

Atravez desses itens temos uma lista ordenada com uma contagem que os precede,que pode ser de qualquer tipo, como numerado, letras, algorismos romanos ou simples simbolos.

Exemplo simples

  1. primeiro item
  2. segundo item
  3. terceiro item

Não Ordenada

O elemento que representa a lista não ordenada é a tag <ul></ul>, seguido de pelo menos um elemento filho denominado <li></li>. Caracteristicamente, os itens na lista desordenada são exibidos com um marcador que pode ter varias formas, e as suas posições na lista é irrelevante.

Exemplo simples:

  • primeiro item
  • segundo item
  • terceiro item
