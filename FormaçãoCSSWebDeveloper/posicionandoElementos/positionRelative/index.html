Posicionando os elementos em tela

A propriedade position vai nos ajudar a manipular a localização/posição do elemento na página.

  • position: static
  • position: relative
  • position: absolute
  • position: fixed
  • position: sticky

Exemplo:

A propriedade position: relative, visualmente vai funcionar igual a propriedade position:static, mas ele faz com que consigamos manipular mais 4 elementos do CSS (top, bottom, left, right). Fazendo com que o elemento se desloque na página.
