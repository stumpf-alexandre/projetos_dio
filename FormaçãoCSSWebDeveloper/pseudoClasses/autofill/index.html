Pseudo-classes :autofill, :webkit-autofill, :-ms-autofill, :-mox-autofill e :-o-autofill

  • :autofill (auto completado pelo navegador)
  • :-webkit-autofill (necessita de um prefixo de navegador para funcionar)
  • :-ms-autofill (necessita de um prefixo de navegador para funcionar)
  • :-mox-autofill (necessita de um prefixo de navegador para funcionar)
  • :-o-autofill (necessita de um prefixo de navegador para funcionar)
