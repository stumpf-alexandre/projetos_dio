Pseudo-elementos

Usado para adicionar elementos extras na página, sem a necessidade de declarã-lo no HTML, ou para selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::first-line

Seleciona a primeira linha de um texto

Propriedades disponiveis:

  • color
  • background
  • font
  • Propriedades de texto
