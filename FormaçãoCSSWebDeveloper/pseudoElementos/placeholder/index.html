Pseudo-elementos

Usado para adicionar elementos extras na página, sem a necessidade de declarã-lo no HTML, ou para selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::placeholder

Um placeholder é uma dica para ajudar no preenchimento de algum campo de formulário

Propriedades aceita:

  • color
  • font
  • background
  • text
