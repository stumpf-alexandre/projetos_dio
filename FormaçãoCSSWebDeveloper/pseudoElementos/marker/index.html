Pseudo-elementos

Usado para adicionar elementos extras na página, sem a necessidade de declarã-lo no HTML, ou para selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::marker

Permite estilizar os tópicos ou números de uma lista li ou então a flecha de um elemento summary

Propriedade aceita:
  • color
  • content
  • white-space
  • font
  • animation
  • transition
