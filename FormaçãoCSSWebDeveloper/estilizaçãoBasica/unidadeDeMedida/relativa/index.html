Unidade de medida em x rem

font-size: 2rem;

O rem tem funcionamento semelhante à unidade de medida em, porém a diferença é que, enquanto o em utiliza o elemento pai como referencia, o rem está relacionado ao tamanho da fonte do elemento raíz, chamado root, que no caso, é a tag html.

font-size: 2em;

A unidade de medida em, quando utilizada para definir o tamanho de fonte de um elemento, vai utilizar como referencia o valor da propriedade font-size do elemento pai no qual está inserido.

Caso seja utilizada a unidade de medida em para outras propriedades que não seja o tamanho da fonte do elemento, o resultado será em relação ao tamanho da fonte do próprio elemento em que está sendo definida a propriedade.

font-size: 200%;

Por que utilizar a unidade de medida rem?

  • Tamanho consistente, sem herança
  • Melhor experiência do usuário

Por que utilizar a unidade de medida em?

  • Contexto dos elementos

Por que utilizar a unidade de medida %?

  • Unudade de medida que irá utilizar como referencia o elemento pai para se basear, podendo alterar os resultados de acordo com o lugar em que está inserido.

Viewport Unit

É a área visível de uma página web, podendo variar de dispositivo para dispositivo. As unidades de medida viewport, vão ser calculadas em relação ao tamanho da área em que o usuário está vendo sua página.

  • Viewport Width vw

  • indica o tamanho da largura da área visível da página.

  • Viewport Height vh

  • indica o tamanho da altura da área visível da página.



width: 50%;
width: 50vw;
width: 50vh;

Unidades de medidas relativas não comumentes usadas

ex

A unidade de medida ex irá usar o tamanho da altura do caracter X (em minúsculo) da fonte do elemento.

O valor especificado altera quando a fonte for completamente alterada.


ch

O ch é uma unidade de medida que possui como referência o caractere 0 da fonte que está sendo utilizada no elemento.

Caso a fonte do elemento seja uma fonte mono-espaçada, ao utilizar a unidade de medida ch, estamos definindo quantos caracteres devem caber no elemento.
