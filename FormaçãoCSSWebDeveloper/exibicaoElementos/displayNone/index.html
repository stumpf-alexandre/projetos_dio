Propriedade Display

A propriedade display vai determinar como essas caixinhas retangulares irão se comportar na hora de serem exibidas na tela.

  • border-width
  • border-style
  • border-color

Exemplo:

display: none;

O display: none esconde o elemento completamente da página, até mesmo os elementos filhos de uma tag
