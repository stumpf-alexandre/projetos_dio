display: flexbox

O flexbox é uma forma de organizar os elementos em uma página dentro de containers e de forma dinâmica, organizando os elementos em uma dimansão: ou horizontalmente (as linhas), ou verticalmente (em colunas).



display: grid

No CSS conseguimos trabalhar o nosso layout utilizando o conteúdo de grids. O grid layout vai organizar os elementos das nossas páginas em duas dimansões: de forma horizontale vertical, definindo linhas e colunas para que os conteúdos da página seja posicionados.
