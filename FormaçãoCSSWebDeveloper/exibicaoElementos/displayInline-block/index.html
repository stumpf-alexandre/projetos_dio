Propriedade Display

A propriedade display vai determinar como essas caixinhas retangulares irão se comportar na hora de serem exibidas na tela.

  • border-width
  • border-style
  • border-color

Exemplo:

display: inline-block;

Os elementos inline-block são parecidos com os elementos inlin, mas agora a largura e a altura dos elementos podem ser alterados



display: inline;
