Hierarquia do Efeito Cascata

Estabelece uma propriedade para aplicação de uma regra de estilo, caso se tenha mais de uma regra para um mesmo seletor CSS.

Se houver um conflito, o navegador irá:

  1. Analisar todas as regras;
  2. Ver o nível de importância na origem do elemento;
  3. Comparar a especificidade das declarações com mesmo nível de importância (entra o calculo de especificidade);
  4. Em caso de empate do grau de especificidade, irá olhar a ordem de declaração colocada dentro dos arquivos de código. A regra CSS que for declarada por último, irá ser selecionada
