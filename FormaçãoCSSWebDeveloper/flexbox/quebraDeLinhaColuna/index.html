Quebra de Linha e de Coluna

Quando o container não consegue organizar seus itens em uma linha, fazendo com que os itens estravaze do container é utilizado comando para poder quebrar a linha e/ou a coluna, para assim organizar melhor os itens no container.

Exemplo: flex-wrap: wrap;

Quando a propriedade flex-wrap vem com o valor wrap os itens do container são organizados para caberem no mesmo.

flex-wrap: wrap;

Em ordem, as linhas são posicionadas da esquerda para a direita, quebrando para uma nova coluna de cima para baixo
