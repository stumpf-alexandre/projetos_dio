Flexbox (Flexible Box Module)

O flexbox é uma forma de organizar os elementos dentro de containers e de forma dinâmica, organizando os elementos em uma dimensão, ou eixo principal (main axis): define a direção em que o nosso container vai organizar os nossos itens, ou horizontalmente (em linhas), ou verticalmente (em colunas). Eixo transversal (cross axis) é perpendicular (atravessa) ao eixo principal.

Os elementos dentro do container ou elemento pai são conhecidos como itens ou elementos filhos.

O flexbox oferece uma maneira mais eficiente de organizar, alinhar e distribuir o espaço entre os itens do container, mesmo que o tamanho seja dinâmico.

Permite trazer para o container a possibilidade de alterar a largura/altura; ordem dos itens e aproveitar o espaço disponível dentro do elemento da melhor forma.
