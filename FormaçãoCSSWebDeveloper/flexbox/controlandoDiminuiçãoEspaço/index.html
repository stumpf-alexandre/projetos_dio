Controlando Diminuição do Item Caso Necessario flex-shrink

A propriedade flex-shrink vai controlar o quanto esse item vai diminuir, caso seja necessário. Se o container não possuir espaço suficiente para acomodar os elementos, eles irão diminuir para caber.

Quanto maior o flex-shrink, menor é o tamanho da área do item.
